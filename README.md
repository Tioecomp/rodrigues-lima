# Rodrigues & Lima Advocacia

Site institucional do escritório Rodrigues & Lima Advocacia e Consultoria.

## Tecnologias

- HTML5 semântico
- CSS3 (Vanilla CSS com variáveis customizadas)
- JavaScript (Vanilla)
- Design System baseado em UI/UX Pro Max

## Estrutura

```
├── index.html          # Página principal
├── css/
│   └── styles.css      # Design system e estilos
├── js/
│   └── main.js         # Navegação e animações
└── assets/
    └── images/         # Imagens do site
```

## Executar localmente

```bash
# Usando npx serve
npx serve .

# Ou usando Python
python -m http.server 8000
```

## Características

- Design minimalista e profissional
- Paleta de cores: Navy Blue + Gold + White
- Tipografia: EB Garamond (headings) + Lato (body)
- Animações suaves ao scroll
- Responsivo (mobile-first)
- Acessível (WCAG)
